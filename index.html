Общая информация:

Данный сайт носит некоммерческий характер, что сказывается на возможностях и функционале:

- в случае отсутствия активности пользователя в течении 6ти месяцев профиль удаляется со всеми данными. Во избежание недоразумений и для возможности восстановления данных рекомендуется делать их Экспорт.

- для корректной работы сайта необходимо использовать современные браузеры.

- расширение функционала не планируется, чтобы сохранить удобство и простоту использования сайта. Однако, здравые идеи по улучшению сервиса приветствуются.

Особенности при работе с полями ввода:
*

- поля обязательные для заполнения

- поля, позволяющие создавать новые объекты или выбирать существующие из раскрывающегося списка, оснащённого функцией быстрого поиска

Работа с примечаниями:

Функционал сайта ограничен 3мя примечаниями для каждого проекта/задачи. При превышении лимита, каждое новое примечание перезаписывает наиболее раннее. При необходимости сохранять всю историю можно переносить Примечания в поле Информации, оформляя в соответствии с собственными потребностями.

Создание и редактирование Стадий и Счётов:

Для создания новой категории Стадий или нового Счёта в соответствующем разделе пункта меню "Настройки" в поле "Наименование" введите новое название и нажмите кнопку "Применить".

Для переименования категории Стадий, Счёта или Шаблона в соответствующем разделе пункта меню "Настройки" в поле "Наименование" выберите название из списка, введите новое наименование в поле "Переименовать", расположенное ниже и нажмите кнопку "Применить".

Использование шаблонов:

Выберите проект, на основании которого требуется создать шаблон. В настройках проекта выберите функцию: "Создать шаблон". Создайте новый проект с использованием ранее созданного шаблона. До настройте пустой проект и перезапишите шаблон с нужными изменениями.
